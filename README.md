# Amazon Sales Data Analysis (Power BI)

## Project Overview
This project analyzes Amazon sales data to identify sales trends, product performance, customer behavior, and geographical distribution using Power BI.

## Tools Used
- Power BI Desktop
- Microsoft Excel
- Microsoft Word

## Project Files
- **Report/**: Detailed analysis report (PDF)
- **Dashboard/**:
  - Power BI dashboard (PDF & screenshots)
  - PBIX file (if supported)

## Key Insights
- Seasonal sales trends observed
- Top-performing product categories identified
- Amazon fulfillment dominates order delivery
- Metro cities contribute highest sales
